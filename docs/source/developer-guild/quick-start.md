## 快速上手

### 搭建开发环境
1. 创建虚拟环境:

    ```shell
    conda create -n AICO python=3.8
    conda activate AICO
    pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
    ```

2. 打开 AICodingOfficer Pro:

    ```shell
    conda activate AICO
    python main.py
    ```
